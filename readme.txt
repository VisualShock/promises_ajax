Установить Node - https://nodejs.org/uk/download/
Открыть консоль и перейти в папку с проектом
Убедиться, что node и npm доступны, выполнить в консоли:
    node -v
    npm -v 
    должны выводить версию
Установить express - npm install express
Запустить сервер - node server.js
Открыть в браузере localhost:4000
Потыкать на кнопки :)

Дополнительно можно поставить фидлер - https://www.telerik.com/download/fiddler
Посмотреть в фидлере на запросы и их содержимое
